# 推荐学习路线

## 必读主线（Agent 构建核心路径）

1. **Ch01 初识智能体** — 建立 Agent 基本概念
2. **Ch04 智能体经典范式构建** — ReAct、Plan-and-Solve、Reflection
3. **Ch07 构建你的 Agent 框架** — 从零实现 HelloAgents
4. **Ch08 记忆与检索** — Memory、RAG、存储
5. **Ch09 上下文工程** — Context Engineering
6. **Ch10 智能体通信协议** — MCP、A2A、ANP
7. **Ch13 或 Ch14 综合案例** — 旅行助手或 DeepResearch
8. **Ch16 毕业设计** — 自己的 Agent 项目

## 次重点（按需深入）

- **Ch03 大语言模型基础** — 需要补 LLM 基础时读
- **Ch06 框架开发实践** — 了解 AutoGen、AgentScope、LangGraph 时读
- **Ch12 智能体评估** — 需要系统评估 Agent 时读

## 可后读（拓展视野）

- **Ch02 智能体发展史**
- **Ch05 低代码平台**
- **Ch11 Agentic RL**
- **Ch15 Cyber Town**

## 每章标准动作

1. 阅读 → 章节笔记
2. 提取 → 概念卡片 / 范式卡片 / 架构卡片
3. 运行 → 代码实验 + 实验记录
4. 记录 → 问题清单
5. 复盘 → 每 3 章一次阶段复盘
