# Ch08 记忆与检索实验记录

## 1. 实验目标

验证 MemoryTool 和 RAGTool 的基本功能，理解记忆系统的四种类型和 RAG 的完整 pipeline。

## 2. 运行环境

- 操作系统：macOS
- Python 版本：3.14.3
- 框架版本：hello-agents 0.2.0
- 外部服务：Qdrant（向量库）、Neo4j（图库）、百炼/本地 Embedding

## 3. 相关文件

- `00_Source/hello-agents/code/chapter8/`（共 11 个示例文件）
- 实验工作空间：`05_实验记录/workspace/ch08_memory_test.py`（待创建）

## 4. 安装和配置

```bash
# 1. 安装框架
pip install "hello-agents[all]==0.2.0"

# 2. 安装 spacy 语言模型
python -m spacy download zh_core_web_sm
python -m spacy download en_core_web_sm

# 3. 在 .env 中配置:
#   - LLM API（OpenAI/ModelScope/智谱等）
#   - Qdrant URL + API_KEY
#   - Neo4j URI + USERNAME + PASSWORD
#   - EMBED_MODEL_TYPE + API_KEY（或切换本地模型）
```

## 5. 实验路线

### 实验 A：MemoryTool 基础操作
运行 `01_MemoryTool_Basic_Operations.py`，验证：
- add（四种类型）
- search
- summary / stats / forget / consolidate

### 实验 B：RAGTool 文档处理
运行 `04_RAGTool_MarkItDown_Pipeline.py`，验证：
- add_document（PDF→Markdown→分块→向量化）
- search（基础检索）
- ask（智能问答）

### 实验 C：高级检索对比
运行 `05_RAGTool_Advanced_Search.py`，对比：
- 基础检索 vs MQE vs HyDE 的召回率差异

### 实验 D：完整 Web 应用
运行 `11_Q&A_Assistant.py`，体验：
- PDF 学习助手的完整功能
- Gradio Web 界面

## 7. 注意事项

- Qdrant 和 Neo4j 有免费云服务额度，首次注册即可使用
- 若无 Embedding API，可切换为本地 `sentence-transformers/all-MiniLM-L6-v2`
- 框架依赖性较重（neo4j, qdrant-client, spacy 等），建议用 `uv` 管理虚拟环境
